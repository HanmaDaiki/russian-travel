# Russian Travel

## Описание

Сайт поделиться идеями для путешествий по России. Реализован с помощью флекс, грид и БЭМ, добавлен фулл адаптив.

## Технологии
  1. HTML
  2. CSS
  3. БЭМ

## Как запустить

```
$ git clone https://github.com/HanmaDaiki/russian-travel
$ cd russian-travel
```

Открыть index.html в браузере.
